A comprehensive, full-stack web application designed to streamline operations for "Salon City". This system manages everything from customer visits and billing to inventory and staff performance, wrapped in a modern, responsive, and aesthetically pleasing interface.

## âš™ï¸ Installation & Setup

Follow these steps to run the project locally.

### Prerequisites
*   **Node.js** (v18 or higher)
*   **npm** (Node Package Manager)

### 1. Clone the Repository
```bash
git clone https://github.com/infinawaz/hair_salon.git
cd hair_salon
```

### 2. Setup Backend (Server)
Navigate to the server directory and install dependencies:
```bash
cd server
npm install
```

Create a `.env` file in the `server` directory with the following variables:
```env
# Database Connection (Use file:./dev.db for SQLite)
DATABASE_URL="file:./dev.db"

# Email Configuration (For sending receipts)
EMAIL_USER="your-email@gmail.com"
EMAIL_PASS="your-app-specific-password"

# Server Port
PORT=3000
```

Initialize the database:
```bash
# Generate Prisma Client
npx prisma generate

# Push schema to database
npx prisma db push

# (Optional) Seed initial data
npm run seed
```

Start the server:
```bash
npm run dev
```
*Server will run on `http://localhost:3000`*

### 3. Setup Frontend (Client)
Open a new terminal, navigate to the client directory, and install dependencies:
```bash
cd client
npm install
```

Start the development server:
```bash
npm run dev
```
*Client will run on `http://localhost:5173`*

## ğŸš€ Deployment

### Production Build
To deploy the application, you typically serve the React frontend through the Express backend.

1.  **Build the Frontend**:
    ```bash
    cd client
    npm run build
    ```
    This creates a `dist` folder.

2.  **Serve Static Files**:
    The backend is configured to serve files from `../client/dist`.

3.  **Start Production Server**:
    ```bash
    cd server
    npm start
    ```

### Render Deployment
This project is configured for deployment on [Render](https://render.com/).
*   **Build Command**: `./render-build.sh`
*   **Start Command**: `npm start`
